# cybersecurity-lab

## Overview
This repository documents my hands-on cybersecurity lab built using Kali Linux, OPNsense, Ubuntu Server, and Windows Server.

## Tools Used
   - Nmap
   - Wireshark
   - Metasploit
   - Virtualbox
   - OPNsense


## Lab Components
   - Network segmentation using OPNsense firewall
   - Active Directory environment (Windows Server)
   - Attack simulations from Kali Linux

## Goals
   - Practice penetration testing techniques
   - Understand network defense mechanisms
   - Simulate real-world attack scenarios
