# Lab Overview

## Purpose
This cybersecurity lab simulates a real-world network environment for practicing penetration testing and defensive security techniques.

## Lab Components
- Kali Linux (Attacker)
- Windows Server (Active Directory & Domain Services)
- Ubuntu Server (Linux services)
- OPNsense (Firewall & Gateway)

## Network Design
- Internal LAN: 192.168.10.0/24
- Gateway: 192.168.10.1 (OPNsense)
- Segmented virtual network using VirtualBox

## Objectives
- Perform network reconnaissance using Nmap
- Enumerate services and users
- Identify vulnerabilities
- Practice controlled exploitation
- Analyze network traffic

## Tools Used
- Nmap
- Wireshark
- Metasploit
- Netcat

## Workflow
1. Reconnaissance
2. Enumeration
3. Exploitation
4. Post-exploitation
5. Documentation

## Notes
This lab is isolated and used strictly for educational purposes.
