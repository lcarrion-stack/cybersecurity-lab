# Engagement Overview - Internal Lab Assessment

## Objective
Simulate a real-world internal penetration test of a virtual network environment.

## Scope
- Windows Server (Active Directory services)
- Ubuntu Server (Linux services)
- OPNsense firewall
- Kali Linux attack machine

## Exclusions
- Host Windows 11 system
- External internet infrastructure

## Rules of Engagement
- No destructive testing
- No attacks outside lab scope
- All findings must be documented with evidence
