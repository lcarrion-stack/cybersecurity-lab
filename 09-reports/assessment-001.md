# Internal Penetration Test Report - Assessment 001

## Executive Summary
Internal lab assessment conducted to evaluate exposed services and network configuration.

## Key Findings
- Windows Server exposed multiple network services
- Internal network hosts are discoverable via scanning
- Firewall rules require validation

## Risk Overview
- Medium overall exposure in lab environment

## Recommendations
- Restrict unnecessary services
- Implement network segmentation rules in OPNsense
- Harden Windows Server services
